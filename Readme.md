## Unity MCP Installation Steps

- Install Python (i suggest homebrew)
- Install UV
- add Unity MCP via package manager (already done in this project)
- Open up Claude code
- Open up the MCP Unity Window (Window > MCP for Unity)
- Start your server
- Configure your client (eg: Claude Code)
    - Run /mcp in claude
    - If the server is not connected, try restarting
